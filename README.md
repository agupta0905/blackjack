Blackjack
=========
A Basic Command line black jack game 

Features
--------------------
* Implemented in C++
* One Player
* One deck
* Deck Shuffled every 6 rounds
* Dealer Hits on Soft 17
* Player Plays First
* Whenever a Participant(Player/Dealer) exceeds 21 he looses

Compiling Instruction
---------------------
* run make
* execute ./blackjack
